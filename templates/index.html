Procedimiento aplicado

Pasos metodológicos seguidos para resolver el problema de transporte.
1
Reconocimiento: Identificar que se trata de un problema de transporte (modelo clásico de Investigación de Operaciones).
2
Levantamiento de datos: Construir la tabla de oferta, demanda y costos unitarios por ruta origen–destino.
3
Verificación de balance: Comparar oferta total vs demanda total para clasificar el problema (balanceado o desbalanceado).
4
Formulación matemática: Definir variables de decisión xij, función objetivo (min Z = Σ cij·xij) y restricciones de oferta y demanda.
5
Método de Vogel: Calcular penalizaciones (diferencia entre los dos menores costos por fila/columna) y asignar iterativamente.
6
Verificación: Confirmar que toda la demanda quedó cubierta y que ningún origen superó su capacidad disponible.
7
Escenario de escasez: Reformular el modelo con variables de déficit y aplicar criterios alternativos de distribución.
8
Comunicación: Traducir la solución matemática a un plan de acción comprensible con tablas, gráficas y visualizaciones.

Modelo matemático completo

// Minimizar costo total de envío min Z = Σᵢ Σⱼ cᵢⱼ · xᵢⱼ // Restricciones de oferta (no enviar más de lo disponible) Σⱼ xᵢⱼ ≤ Sᵢ ∀ i ∈ {Manizales, Pereira, Armenia} // Restricciones de demanda (cubrir toda la necesidad) Σᵢ xᵢⱼ ≥ Dⱼ ∀ j ∈ {Sarajevo, Ucrania, Polonia} // No negatividad xᵢⱼ ≥ 0 ∀ i, j // Caso escasez: añadir variable de déficit sⱼ Σᵢ xᵢⱼ + sⱼ = Dⱼ (sⱼ ≥ 0) min Z = Σ cᵢⱼ·xᵢⱼ + Σ P·sⱼ (P = penalización por déficit)
